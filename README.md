In the new "Next-Gen" Roblox Studio UI, Roblox allows for creation of new tabs and items in those tabs for some weird reason they just don't include some important items in the avaliable items list (Part dropdown, Part grid, Part rotation, etc) making it very inconvenient, on top of the new UI just having the weirdest layout ever it causes annoyances.

I dispise the new UI so much to the point I've spent hours coding in JSON to change every tab, and the items in the tab to match old Roblox Studio UI. This isn't a 1:1 recreation since Roblox limitations but it's pretty close ­şæı

From personal experience Roblox Studio updates *shouldn't* reset the custom tabs, but Roblox being Roblox some update will probably delete the custom tabs, so keep that in mind.
